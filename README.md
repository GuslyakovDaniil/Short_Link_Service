# Сервис Сокращения Ссылок (Link Shortener)

Проект представляет собой консольный сервис на Java для создания коротких ссылок с возможностью установки лимита переходов и времени жизни.

## Архитектура
Проект разработан с использованием слоеной архитектуры для разделения ответственности:
- **CLI (`cli`):** Пользовательский интерфейс в консоли.
- **Service (`service`):** Слой, содержащий основную бизнес-логику.
- **Repository (`repository`):** Слой доступа к данным (реализован in-memory).
- **Scheduler (`scheduler`):** Фоновый процесс для очистки просроченных ссылок.
- **Config (`config`):** Модуль для чтения настроек из `config.properties`.

## Как запустить
1. Склонируйте репозиторий: `[git clone https://github.com/ВАШ_НИК/link-shortener.git](https://github.com/GuslyakovDaniil/Short_Link_Service.git)`
2. Перейдите в папку проекта: `cd link-shortener`
3. Соберите проект с помощью Maven: `mvn clean package`
4. Запустите приложение из командной строки: `java -jar target/link-shortener-1.0-SNAPSHOT.jar`

## Команды
После запуска введите одну из следующих команд:

- `create <длинный_url> [лимит_переходов]` — Создает короткую ссылку. Если лимит не указан, используется значение по умолчанию.
  *Пример:* `create https://www.google.com 10`
- `open <короткий_код>` — Открывает исходную ссылку в браузере.
  *Пример:* `open aB1cD2e`
- `delete <короткий_код>` — Удаляет вашу ссылку (требует UUID, полученный при создании первой ссылки).
  *Пример:* `delete aB1cD2e`
- `list` — Показывает список всех ваших созданных ссылок.
- `help` — Показывает это сообщение.
- `exit` — Выход из приложения.
